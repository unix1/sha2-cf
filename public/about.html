sha2.dev

What

sha2. returns sha2 hash of whatever you ask it to. That's it! Send any content and get its sha2 hash.

sha2. is a fully open source project that anyone can contribute to.

Why

Just to serve as a simple example for anyone getting started with Cloudflare Pages.

How

Technical details:

  • Code is in a simple Cloudflare Pages app written in plain JavaScript on the frontend and TypeScript on the backend.

Source code:

  • Source code for sha2.dev web application on GitHub https://github.com/unix1/sha2-cf

Contribute

Contributions are encouraged. Feel free to submit an issue or a PR.
